# Redis Clone in Go

> A tiny Redis-like server built in Go for the Codecrafters challenge.

## ✨ Highlights

- RESP parsing (`*array`, `$bulk string`)
- Commands: `PING`, `ECHO`, `SET`, `GET`, `SET EX`, `SET PX`
- Expiration handling and integration tests

## 🚀 Quick Run

```sh
go run ./app
```

## ✅ Quick Test

```sh
go test ./tests -v
```
